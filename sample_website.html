// How It Works

MULTI-
SIGNAL
PIPELINE

Every package runs through five independent detection layers in parallel, cross-referenced against our threat graph of 50M+ known package relationships and maintainer identities.

01

Static Analysis

AST scanning for obfuscated code, eval() abuse, suspicious string patterns, and exfiltration signatures.

02

Behavioral Sandbox

Runtime execution in isolated containers monitoring network calls, filesystem access, and spawned processes.

03

Typosquatting AI

ML model trained on 8M+ package names detecting character substitution, look-alike, and homoglyph attacks.

04

Graph Intelligence

Dependency graph analysis tracing malicious lineage, hijacked maintainers, and deep transitive threats.

05

Threat Feeds

Cross-referenced against CVE databases, OSS advisories, and Vexor's proprietary zero-day threat intelligence.
